Cara Upload Gambar dan File di Laravel 4

Syaiful Bahri

Published 21 Jul 2014

Laravel dibangun dengan beberapa dependency, salah satunya adalah Symfony Http Foundation. Cara upload file dan gambar di Laravel 4, akan mengambil atau meminjam komponen dari Symfony.

Saya akan membuat sebuah form sederhana, dengan menggunakan Blade.

<!-- app/views/upload-form.blade.php -->

<form action="{{ url('post-upload') }}"
  method="POST"
  enctype="multipart/form-data">

 <input type="file" name="image" />

 <input type="submit">
</form>

Oke. Kita telah mempunyai form sederhana untuk membuat pengguna kita bisa mengumpload file ke server kita. Ketika Anda ingin mengupload file pastikan Anda memberi atribute multipart/form-data. Tanpa itu, upload file tidak akan bekerja.

Sekarang Anda bisa membuat route, untuk menampilkan form tersebut.

Route::get('/upload', function()
{
    return View::make('upload-form');
});

Coba Anda mengunjungi route /upload, dan Anda akan melihat form kita telah tampil. Itu bagus, jadi mari kita upload gambar. Mungkin gambar yang kita upload salah satu foto saya, nama filenya syaiful.jpg.

Jika Anda memperhatikan form yang saya buat di atas, maka anda tahu form tersebut akan mengirim request ke route /post-upload dengan request method POST.

Jadi kita harus membuat route tersebut.

Route::post('/post-upload', function()
{
    $file = Input::file('image');

    var_dump($file);
});

Jika Anda mencoba upload maka Anda akan melihat informasi objek yang kita dump tadi.

Obyek tersebut adalah instance dari class Symfony\Component\HttpFoundation\File\UploadedFile. Untungnya class tersebut sangat mudah digunakan untuk menghandle upload file. Dan yang paling menarik, class tersebut inherit dengan SplFileInfo Silakan Anda membaca class tersebut di folder vendor Anda.

Untuk membuat tutorial ini simple, saya akan memberi Anda contoh bagaimana menggunakan objek tersebut.

Route::post('/post-upload', function()
{
    $file = Input::file('image');
    /** 
    * ambil name file asli file, bukan nama
    * file yang dibuat php di tmp folder
    */
    $name = $file->getClientOriginalName();

    //pindah file dari tmp ke folder asset
    $file->move('/path/to/storage', $name);

});

Cobalah untuk mengupload gambar syaiful.jpg ke aplikasi kita. Jika Anda melihat file syaiful.jpg ada di path /path/to/storage/ maka Anda berhasil. Jadi selamat!

Begitulah cara mengupload gambar atau file di Laravel 4. Sangat mudah dan simple.
